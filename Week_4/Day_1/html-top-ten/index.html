HTML

Here are a few HTML things I like:

EMBEDDED AUDIO TAGS

The < audio > tag takes a relative address or URL and embeds it into the webpage. It has a number of attributes to allow for automatic playback, controls, and even initial audio levels and looping. Currently, I have it working from a local directory. If you want to pull from a URL, then you'll have to have the direct link to the .mp3, rather than just the page the song is hosted on.

Link to audio tag usage

EMBEDDED VIDEOS

The < video > tag allows for embedded videos! Just like with audio, you can use a relative address but a URL that directly leads to the .mp4 or .webm/webp file needs to be configured with MIME properties. Relative addressing will allow you to load straight from source.

Link to video tag usage

PAGES IN PAGES

Using < iframe >, you can embed a webpage/video within another page! There are limitations, however. One of these is x-source security, which will flat out prevent iframe from framing the source to prevent click abuse. If the embedded page has their own set of iframes (as shown below), then unexpected behavior will likely occur!
